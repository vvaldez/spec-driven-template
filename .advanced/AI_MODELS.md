# AI_MODELS.md

## Purpose
Document which AI tools you use, what each is good at, and how they should interact with this template.

## Model and Tool Catalog
For each model or tool, note:
- **Name** (e.g., Claude, ChatGPT, Gemini, Copilot, Cursor, Gemini Code Assist, Codeium).
- **Role** (chat planning, IDE assistant, research, etc.).
- **Strengths**.
- **Things to watch out for**.

## When to Use Which Tool
Describe typical scenarios and which tools you prefer for each, such as:
- Brainstorming and high-level planning.
- Editing and organizing specs.
- Researching prior art or related patterns.

## Safety and Boundaries
Reinforce any special safety notes or constraints for particular tools (for example, where data residency or privacy matters).
